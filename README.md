## 🚀 Main Projects

### 1. SYNK — 실시간 동기화 미션 서비스

> **“우리는 지금, 같은 순간을 산다.”**

친구들이 같은 시간에 랜덤 미션을 수행하고, 짧은 영상을 제출하면 하나의 콜라주로 공유하는 실시간 동기화 서비스입니다.

* **기간**: 2026.05 ~ 진행 중
* **팀 구성**: 2인 팀 / 백엔드 담당
* **주요 역할**

  * Spring Boot 기반 REST API 설계 및 구현
  * WebSocket 기반 실시간 미션 이벤트 처리
  * Spring Scheduler 기반 미션 생성·활성화·마감 자동화
  * AWS Lambda, S3, FFmpeg 기반 영상 콜라주 비동기 처리
  * AWS EC2, Docker, GitHub Actions 기반 배포 환경 구성

🔗 **Service**: https://synk-front.vercel.app/

### 3. AdProb — 머신러닝 기반 블로그 광고 탐지 시스템

블로그 리뷰가 광고성인지 판단하기 어려운 문제를 해결하기 위해, 텍스트와 이미지를 함께 분석하는 광고 탐지 시스템을 개발했습니다.

* **기간**: 2025.03 ~ 2025.06
* **기술 스택**: Python, Flask, JavaScript, CatBoost, LSTM AutoEncoder, OCR, GPT API
* **주요 역할**

  * Chrome Extension 기반 블로그 본문 및 이미지 데이터 수집
  * Flask 서버 기반 데이터 수신 및 전처리
  * CatBoost 모델 기반 1차 광고 확률 예측
  * LSTM AutoEncoder 기반 애매한 데이터 재판별
  * 광고 여부와 근거 문장을 보여주는 웹 대시보드 구현
* **성과**

  * 규칙 기반 탐지 방식 대비 정확도 65% → 89% 향상
